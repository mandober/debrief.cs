# Computer hardware

https://en.wikipedia.org/wiki/Computer_hardware

Computer hardware comprises physical parts of a machine including, at least, a motherboard with the CPU and some memory modules installed, and a PSU; boot off the network and controlled (I/O) from another device. Local control requires a keyboad and a monitor, permanent storage (SDD, HDD) preferable.
