## 2's complement

flipSign(n) = compl(n) + 1
n ⟼ -n
-n ⟼ n, unless -n is MAX_NEG, since |neg| = |pos| - 1, `-2ⁿ⁻¹` .. `2ⁿ⁻¹-1`
